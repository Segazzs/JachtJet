# YachtJet

YachtJet is a responsive interactive landing page for showcasing yacht rental services. The project is created according to a ready-made layout and includes a main page structure, a yacht list section, blocks about benefits, a rental form and reviews.

Technologies

The project was implemented using:

HTML — structural page markup;

CSS — component styling and responsive layout;

JavaScript — interactivity for the mobile card section and modal window.

Implementation features

The Cards / Fleet section is adapted to mobile devices: by default, some of the cards are cropped, and when interacting, they expand with animation.

The mobile menu and modal windows are implemented in JavaScript for better user interaction.

The layout is made according to the finished design, taking into account typography and style.

Adaptability to different screen sizes is ensured using media queries.

Structure

The project contains the following conditional sections:

Header with navigation;

Hero block with title and subtitle;

Section with a list of yachts (Cards);

“Plus” block with a list of advantages;

Rental section with a data entry form;

Reviews block;

Footer with navigation links.

Interactivity

JavaScript is used for:

Controlling the expansion of the card section on mobile;

Opening and closing the modal menu;

Controlling the state of the interface depending on user actions.
